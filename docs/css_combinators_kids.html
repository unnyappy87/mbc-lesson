👨‍👩‍👧‍👦 CSS 관계 마법: 자식과 형제

웹사이트의 요소들도 우리처럼 가족과 친구 관계가 있어요!
어떻게 관계를 맺어 마법을 부리는지 자세히 알아봐요.

> 1. 자식 선택자

💡 비유: "엄마 바로 옆에 있는 아들만!"

어떤 요소의 바로 아래에 있는 '직계 자식'만 콕 집어서 꾸며요.

  • 모양: 부모와 자식 사이에 >를 써요.
  • 범위: 손자, 증손자는 안 돼요! 오직 바로 밑의 자식만 해당돼요.
div > p { color: red; } /* div 안에 바로 있는 p만 빨갛게! */

(띄어쓰기) 2. 후손 선택자

💡 비유: "우리 가문 모든 사람들 다 모여라!"

어떤 요소 안에 살고 있는 모든 자식, 손자, 증손자들을 전부 다 꾸며요.

  • 모양: 이름 사이에 한 칸 띄어쓰기를 해요.
  • 범위: 깊숙이 숨어있는 후손들까지 모두 마법에 걸려요.
div p { color: blue; } /* div 안에 살고 있는 모든 p를 파랗게! */

+ 3. 인접 형제 선택자

💡 비유: "내 바로 뒤에 줄 서 있는 친구 한 명만!"

어떤 요소의 바로 뒤에 나오는 형제 딱 한 명만 꾸며요.

  • 모양: 이름 사이에 +를 써요.
  • 약속: 형제여야 하고(부모가 같음), 반드시 바로 옆에 딱 붙어 있어야 해요.
h1 + p { font-weight: bold; } /* h1 바로 다음에 오는 p만 굵게! */

~ 4. 일반 형제 선택자

💡 비유: "내 뒤에 줄 서 있는 모든 동생들 다 같이!"

어떤 요소 뒤에 나오는 모든 형제 동생들을 한꺼번에 꾸며요.

  • 모양: 이름 사이에 ~(물결)을 써요.
  • 범위: 바로 옆이 아니어도 괜찮아요. 뒤에 있기만 하면 다 꾸며줘요.
h1 ~ p { color: green; } /* h1 뒤에 나오는 모든 p를 초록색으로! */

📖 마법 공식 정리

기호 마법 이름 대상
A > B 자식 선택자 A의 바로 밑 자식인 B
A B 후손 선택자 A 안에 있는 모든 B (손자 포함)
A + B 인접 형제 선택자 A의 바로 뒤에 오는 형제 B (딱 1개)
A ~ B 일반 형제 선택자 A의 뒤에 오는 모든 형제 B들
⭐ 팁: "자식"과 "후손"의 차이를 잘 기억하면 나중에 아주 멋진 웹사이트를 만들 수 있어요!
