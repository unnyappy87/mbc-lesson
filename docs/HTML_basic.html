📄 HTML 기본 문서 구조

처음 배우는 친구들을 위한 쉬운 설명 + 상세 정리

🏠 HTML은 집과 같아요!

HTML 문서를 집에 비유하면 쉽게 이해할 수 있어요.

🚪 집 전체 = <html>
집 안에는 방이 여러 개 있죠? HTML도 마찬가지예요.
문서 전체를 <html>이라는 큰 집으로 감싸요.

📋 1. <!DOCTYPE html> — "나는 HTML5 문서야!"

이건 문서 맨 위에 쓰는 이름표예요.

브라우저(크롬, 사파리 등)에게 "이 파일은 HTML5로 만들었어!"라고 알려주는 거예요.

💡 비유: 학교에 입학할 때 "저는 1학년이에요!"라고 말하는 것과 같아요.

🧠 2. <head> — 머리(눈에 안 보이는 정보)

<head>는 화면에 보이지 않는 정보를 담는 방이에요.

  • <title> — 브라우저 탭에 나오는 제목
  • <meta charset="UTF-8"> — 한글을 제대로 보여주는 설정
  • <meta name="viewport"> — 휴대폰에서도 잘 보이게 하는 설정
💡 비유: 집의 전기 설비, 수도 설비 같은 거예요.
눈에 안 보이지만 없으면 집이 제대로 작동하지 않죠?

🖥️ 3. <body> — 몸(눈에 보이는 내용)

<body>는 웹 페이지에서 실제로 보이는 모든 것을 담는 방이에요.

  • <h1> — 큰 제목 (책 표지 같은 거)
  • <p> — 글 단락 (책 속 내용)
  • <img> — 사진
  • <a> — 링크 (누르면 다른 페이지로 이동)
💡 비유: 집의 거실, 침실, 주방 같은 거예요.
사람들이 실제로 보고 사용하는 공간이죠!

📦 전체 모습 한눈에 보기

<!DOCTYPE html>           <!-- "나는 HTML5야!" -->
<html>                      <!-- 집 전체 -->
    <head>                  <!-- 머리 (눈에 안 보이는 정보) -->
        <meta charset="UTF-8">
        <title>나의 첫 웹페이지</title>
    </head>
    <body>                  <!-- 몸 (눈에 보이는 내용) -->
        <h1>안녕하세요!</h1>
        <p>저는 HTML을 배우고 있어요.</p>
    </body>
</html>                     <!-- 집 끝 -->
🎯 핵심 정리

<!DOCTYPE> → 이름표
<html> → 집 전체
<head> → 머리 (안 보이는 정보)
<body> → 몸 (보이는 내용)

이 4가지만 기억하면 HTML의 뼈대는 끝!
✦ ✦ ✦

📚 상세 설명

HTML 문서의 기본 구조는 다음과 같아요.

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>문서 제목</title>
</head>
<body>
    <!-- 실제 화면에 보이는 내용 -->
</body>
</html>

🔍 각 태그의 역할

  • <!DOCTYPE html> — 브라우저에게 "이 문서는 HTML5로 작성되었다"고 알려주는 선언문이에요. 맨 위에 한 줄만 적어주면 돼요.
  • <html> — HTML 문서 전체를 감싸는 최상위 태그예요. lang="ko" 속성으로 문서의 주요 언어를 지정할 수 있어요.
  • <head> — 화면에 직접 보이지 않지만, 문서에 필요한 여러 정보를 담는 공간이에요.
  • <body> — 사용자가 실제로 보고 상호작용하는 모든 콘텐츠를 담는 공간이에요. 텍스트, 이미지, 링크, 표, 버튼 등 모든 시각적 요소가 여기에 들어가요.

🧠 <head> 안에 들어가는 주요 태그

  • <meta charset="UTF-8"> — 문자 인코딩 방식을 지정해요. 한글을 제대로 표시하려면 필수예요.
  • <meta name="viewport" ...> — 모바일 화면에서도 적절한 크기로 보이도록 설정해요.
  • <title> — 브라우저 탭에 표시되는 문서 제목이에요.
  • <link rel="stylesheet" href="..."> — 외부 CSS 파일을 연결할 때 써요.
  • <script src="..."></script> — 외부 JavaScript 파일을 불러올 때 써요.

🖥️ <body> 안에 들어가는 주요 태그

  • <h1> ~ <h6> — 제목(헤딩) 태그예요. 숫자가 작을수록 더 중요한 제목이에요.
  • <p> — 단락(문단)을 만들어요.
  • <a href="..."> — 하이퍼링크(다른 페이지로 이동)를 만들어요.
  • <img src="..." alt="..."> — 이미지를 삽입해요.
  • <div> — 콘텐츠를 그룹으로 묶는 용도의 블록 컨테이너예요.
  • <span> — 텍스트의 일부분만 스타일을 적용할 때 쓰는 인라인 컨테이너예요.
💡 핵심 요약
HTML 문서는 <!DOCTYPE> 선언 → <html> 루트 → <head>(메타정보) → <body>(화면 내용) 순서로 구성돼요.
이 다섯 가지가 웹 페이지의 가장 기본이 되는 뼈대예요.
