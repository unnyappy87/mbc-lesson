Gemini CLI 런처 사용법

파일 위치: C:\Users\PC\Desktop\beep\web\html\

방법 1: 우클릭 메뉴에 추가 (강력 추천)

폴더를 오른쪽 클릭하기만 하면 해당 위치에서 제미나이가 실행됩니다.

  1. 위 폴더에서 add_context_menu.reg 파일을 찾습니다.
  2. 파일을 더블 클릭하여 실행합니다.
  3. "레지스트리 편집기" 경고창이 뜨면 [예]를 클릭합니다.
  4. 설정이 완료되었습니다! 이제 아무 폴더에서나 바탕화면 빈 곳을 마우스 우클릭하면 Open Gemini CLI here 메뉴가 보입니다.

방법 2: 배치 파일로 직접 실행

특정 위치에 파일을 두고 더블 클릭하여 실행하는 방식입니다.

  1. gemini_launcher.bat 파일을 제미나이를 실행하고 싶은 폴더로 복사합니다.
  2. 복사한 파일을 더블 클릭합니다.
  3. 검은색 창(명령 프롬프트)이 뜨면서 제미나이 CLI가 시작됩니다.
💡 팁:
파일이 보이지 않는다면 윈도우 탐색기의 주소창에 C:\Users\PC\Desktop\beep\web\html 를 복사해서 붙여넣고 엔터를 치시면 바로 이동할 수 있습니다.
