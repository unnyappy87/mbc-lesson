✨ CSS 선택자 마법 사전

웹사이트의 요소를 콕 찍어서 꾸미는 마법의 주문(선택자)을 배워봐요!

p, h1, div 태그 선택자

💡 비유: "이 세상의 모든 사과 나와라!"

특정한 종류의 태그를 가진 모든 친구들을 한꺼번에 꾸미고 싶을 때 사용해요.

  • 사용법: 그냥 태그 이름을 그대로 써요.
  • 특징: 페이지에 있는 똑같은 태그는 모두 다 변해요.
p { color: blue; } /* 모든 글자를 파랗게! */

. 클래스 선택자

💡 비유: "노란색 티셔츠 입은 사람만 모여라!"

서로 다른 태그라도 내가 정한 '별명'을 가진 친구들만 골라서 꾸밀 때 써요.

  • 사용법: 이름 앞에 점(.)을 찍어요.
  • 특징: 한 명도 되고, 열 명도 되고, 여러 번 쓸 수 있어요!
.yellow-team { background: yellow; }

# 아이디 선택자

💡 비유: "우리 반 1번 홍길동 딱 한 명만!"

이 세상에 딱 하나뿐인 아주 특별한 부분을 콕 집어서 꾸밀 때 써요.

  • 사용법: 이름 앞에 샵(#)을 붙여요.
  • 특징: 한 페이지에 딱 한 번만 쓸 수 있어요. 가장 힘이 세요!
#main-logo { width: 200px; }

, 그룹 선택자

💡 비유: "사과랑 포도랑 딸기 다 같이 빨갛게!"

서로 다른 친구들에게 똑같은 스타일을 한 번에 주고 싶을 때 사용해요.

  • 사용법: 선택자들 사이에 쉼표(,)를 찍어요.
  • 장점: 코드를 여러 번 안 써도 되니까 아주 편리해요!
h1, h2, .title { color: red; }

정리해볼까요?

마법 종류 표시 방법 누구를 꾸미나요?
태그 선택자 태그이름 그 태그를 쓴 모든 친구들
클래스 선택자 .이름 같은 별명을 가진 여러 친구들
아이디 선택자 #이름 딱 하나뿐인 주인공 친구
그룹 선택자 A, B A랑 B 모두 다 같이

🌈 마법사의 마지막 꿀팁!

보통은 클래스(.)를 가장 많이 써요. 아이디(#)는 정말 특별한 친구에게만 아껴서 사용하고, 비슷한 스타일이 많으면 쉼표(,)로 묶어서 마법을 부려보세요!
