✨ CSS 변신 마법: 클래스와 아이디

. 1. Class (클래스) - "우리 반 단체복"

클래스는 여러 친구들에게 똑같은 스타일을 줄 때 사용해요.

  • 비유: 우리 반 친구들이 모두 입는 '체육복'과 같아요.
  • 특징: 한 페이지에 몇 번이고 다시 쓸 수 있어요. 한 명이 여러 벌의 옷(클래스)을 입을 수도 있죠!
  • 약속: 이름 앞에 꼭 점(.)을 찍어주세요.

# 2. ID (아이디) - "나만의 이름표"

아이디는 이 세상에 딱 하나뿐인 부분을 꾸밀 때 사용해요.

  • 비유: 내 가방에 붙은 하나뿐인 '이름표'나 내 '지문'과 같아요.
  • 특징: 한 페이지에 딱 한 번만 쓸 수 있어요. 중복되면 컴퓨터가 헷갈려해요!
  • 약속: 이름 앞에 꼭 샵(#)을 붙여주세요.
